# Whisper Homebrew tap

```sh
brew install whisper-sec/tap/whisper
```

The [`whisper`](https://github.com/whisper-sec/whisper-cli) CLI gives any agent a
routable Whisper IPv6 identity and safe egress. The binaries are the signed
release artifacts from `whisper-cli` (verify with the published PGP key).

```sh
whisper --help
```
